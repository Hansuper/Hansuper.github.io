## 快速计算是否成年 <!-- {docsify-ignore} -->

> 需求背景：假如小明出生在2008年8月8日、在数据库中存储的时间戳 1218124800，需要判断小明截止今日是否成年

如果按照我们常规的想法伪代码处理如下

```
time() - 1218124800 > 18 * 86400 * 365
```

以上代码虽然能判断是否成年，但不够优雅和准确，因为假如中间有闰年的存在就不是365天了

更优雅的办法

```
xiaoming_birthday = date('Ymd', 1218124800) #出生的年月日
cur_day = date('Ymd') #当前的年月日

if(xiaoming_birthday + 180000 < cur_day){
    echo '未成年';
}else{
    echo '已成年';
}

# 相当于我们直接在年份上加18来判断是否小于当前的年月日

```
